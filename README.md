
这个项目存在的目的，是测试“依附的自由”理论能否被所有人方便的应用。

github是这个理论的著名案例。全站强制HTTPS链接，因此在这里你的访问是较为安全的，GFW不能针对性地屏蔽某个网站——要么视而不见，要么屏蔽整个github。由于这个代码托管网站对于中国的程序员是那么的重要，以至于GFW不敢轻易屏蔽这个网站，于是大陆网民都可以访问依附于本站的所有页面。
